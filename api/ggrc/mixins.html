Categorizable¶

class ggrc.models.categorization.Categorizable¶

Subclasses MUST provide a declared_attr method that defines the relationship and association_proxy. For example:

@declared_attr
def control_categorizations(cls):
  return cls.categorizations(
      'control_categorizations',
      'control_categories',
      100,
  )

Mixin for commentable objects.

This is a mixin for adding default options to objects on which people can comment.
